# Nineti-Assignment-Solution
Nineti Assignment - How to transfer SSID and Password through Bluetooth Low Energy (BLE) using ESP BLE Provisioning app, to connect ESP32 to WiFi or Hotspot

# Requirements
1) ESP32 Dev Board 

2) ESP-IDF 

3) WiFi availability
